# NexonOS

NexonOS is a Linux distribution built on **Arch Linux**, designed to be fast, simple, and user-friendly. It comes with **KDE Plasma** as the default desktop environment, pre-configured for both productivity and gaming.

## Features

- **Custom Installer:** A straightforward installer that guides you through setup quickly.
- **Pre-installed Software:** Includes Firefox, Discord, Thunderbird, Yay, and more.
- **Gaming Ready:** Steam comes pre-installed and optimized for most games.
- **GPU Setup Tool:** Install NVIDIA, AMD, or Intel drivers easily with a guided tool. ( SOON )
- **System Tools:** Downgrade packages, manage updates, and install software without hassle.
- **Lightweight & Fast:** Minimal bloat while keeping essential functionality.

## Why NexonOS?

NexonOS aims to provide a ready-to-use Arch Linux experience without the complexity of manual configuration. Whether you're gaming, browsing, or working, it comes pre-configured to save you time.

## Customization

KDE Plasma offers full flexibility for users to customize their desktop experience. Themes, layouts, and widgets can be changed with a few clicks, making NexonOS adaptable to any workflow.

## Beginner-Friendly

While NexonOS is based on Arch Linux, it is designed to be approachable for new Linux users. The custom installer, pre-installed applications, and GPU setup tool reduce the learning curve significantly.

## Advanced Users

Power users can still take advantage of Arch Linux's flexibility. You can use `yay` for AUR packages, perform downgrades, and manage your system as you would on any Arch-based distribution.

## Build From Source

You can create your own NexonOS image using ArchISO or other Arch-based build tools. Development builds may be less stable than official releases, but allow full customization.
